## Run the Application
**A. Setup & Build**
1. Open the Terminal and navigate to the project folder.

2. Install basic environment:

    ```bash
    npm install
    ```
    

**B. Launch Server**
1. Run the following command to start the local server:

    ```bash
    npm start 
    ```

**C. View Webpage**
* **On Computer**   

    Open a web browser (Chrome/Safari) and navigate to http://localhost:3000
* **On Mobile Phone**   

    Ensure the mobile device and computer are on the SAME Wi-Fi network.    
    Find the computer's Local IP address, and navigate to http://[YOUR_IP]:3000 on the mobile browser.
    ###### Example: http://192.168.1.16:3000


## Testing for Majority of the Users
To ensure the web application works for the majority of users, I 
* Manually resize the browser windows on the computer to verify that the CSS Grid layout adapts correctly.
* Test the application on two major browsers, Google Chrome and Safari, to ensure cross-browser compatibility.
* Access the local server via computer's IP address on my mobile phone over the same Wi-Fi to test the actual mobile user experience.
