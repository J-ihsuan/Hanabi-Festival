## Run the Application
**A. Setup & Build**
1. Open the Terminal and navigate to the project folder.

2. Install the required dependencies (TypeScript):

    ```bash
    npm install
    ```
3. Compile the TypeScript file `main.ts` into standard JavaScript `main.js`:

    ```bash
    npx tsc main.ts
    ```

**B. Launch Server**
1. Run the following command to start the local server:

    ```bash
    npm start 
    ```

**C. View Webpage**
* **On Computer**   

    Open a web browser (Chrome/Safari) and navigate to http://localhost:3000
* **On Mobile Phone**   

    Ensure the mobile device and computer are on the SAME Wi-Fi network.    
    Find the computer's Local IP address, and navigate to http://[YOUR_IP]:3000 on the mobile browser.
    ###### Example: http://192.168.1.16:3000

## TypeScript Features Used in This Assignment
**Classes**
* Define a `HanabiEvent` class to act as the object for the festival data. Use a `constructor` method to initialize properties.

**Basic Types**
* Enforce strict typing for variables and class properties (e.g., `string`, `number`, and `HanabiEvent[]`) to ensure data consistency and catch potential errors.

**Arrow Functions**
* Use arrow functions (`=>`) for the rendering logic and array iteration methods `.forEach()`.

**Template Literals**
* Replaced traditional string concatenation with ES6 template literals `` ` `` to inject TypeScript variables into the HTML markup string (e.g., `${event.title}`).
